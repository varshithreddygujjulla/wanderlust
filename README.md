# Wanderlust 🌍

Wanderlust is a full-stack travel listing web application where users can explore, create, edit, and delete travel listings.

## Features
- View all travel listings
- View details of a listing
- Add new listings
- Edit existing listings
- Delete listings
- Form validation using Joi
- Responsive UI using Bootstrap

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap
