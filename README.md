# Wanderlust 🌍

Wanderlust is a full-stack travel listing web application where users can explore, create, edit, and manage travel listings.

## Features
- View all travel listings
- View details of a listing
- Add new listings
- Edit existing listings
- Delete listings
- Form validation using Joi
- Responsive UI using Bootstrap
- RESTful routing implementation

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap

## Project Structure
models/
views/
public/
utils/
init/
app.js
schema.js

## Installation

1. Clone the repository

git clone https://github.com/varshithreddygujjulla/wanderlust.git

2. Install dependencies

npm install

3. Initialize database

node init/index.js

4. Start the server

node app.js

5. Open in browser

http://localhost:8080/listings
